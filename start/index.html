安装（快速）¶

步骤一：飞前

Ceph 客户端和 Ceph 节点需要些基本预配置才能部署 Ceph 存储集群，你也可以加入 Ceph 社区以寻求帮助。

  • 飞前
    • Ceph 部署工具的安装
      • 高级包管理工具（APT）
      • 红帽包管理工具（RPM）
    • Ceph 节点安装
      • 安装 NTP
      • 安装 SSH 服务器
      • 创建部署 Ceph 的用户
      • 允许无密码 SSH 登录
      • 引导时联网
      • 确保联通性
      • 放通所需端口
      • 终端（ TTY ）
      • SELinux
      • 优先级或首选项
    • 总结

步骤二：存储集群

完成飞前检查表之后，应该可以开始部署 Ceph 存储集群了。

      • 添加 RGW 例程
